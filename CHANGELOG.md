# Changelog

## 0.1.0 - 2026-03-09

- added a macOS CUPS filter for the Fuji Xerox DocuPrint M205 b
- added a reusable PPD template for print-only support
- vendored the upstream `foo2hbpl2` encoder sources needed to build locally
- added a root-only installer for a normal CUPS queue
- added a helper path for raw HBPL submission
- documented licensing and upstream attribution
- added a reusable validation script and GitHub Actions CI workflow
- prepared the project for public release as source
