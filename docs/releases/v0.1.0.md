# v0.1.0

Initial public source release of the DocuPrint M205 b macOS driver prototype.

## Highlights

- print-only support for the Fuji Xerox DocuPrint M205 b
- macOS CUPS filter that generates monochrome HBPL
- reusable PPD template
- root-owned installer path for a normal queue
- helper script for raw HBPL submission
- local validation script and GitHub Actions CI workflow

## Known limitations

- no scanning support
- the helper queue is only for raw helper submissions
- the normal installed queue still depends on the legacy CUPS driver model that
  Apple has deprecated
